## 1. 项目概述

sift-kg 是一个**零配置的文档到知识图谱转换管道**，可以将任意文档集合（PDF、DOCX、HTML、扫描件等75+格式）转换为可浏览、可查询的知识图谱。其核心特点是：

- **任何人可用**: 不需要数据库、不需要基础设施，只需要一个CLI工具
- **LLM 提取**: 使用大语言模型提取实体和关系，支持任意LLM提供商（OpenAI、Anthropic、Mistral、Ollama本地模型等）
- **人机协作**: LLM提出实体合并建议，人工审核批准，不自动合并任何内容
- **灵活模式**: 支持预定义schema（固定实体/关系类型）和schema-free（LLM自动从语料发现schema）两种模式
- **增量处理**: 只重新提取修改过的文档，缓存提取结果节省成本
- **可作为AI第二大脑**: 生成的知识图谱可以直接作为AI agent的结构化记忆，提供持久化的知识理解

**主要用例**:
- 研究与教育：文献综述概念图、课程知识梳理
- 商业情报：竞争对手白皮书、市场报告分析
- 调查工作：FOIA文件、法庭文件、公开记录分析
- 法律审阅：多文档实体关系抽取
- 族谱：家谱关系追踪

## 2. 架构概览

sift-kg 采用模块化流水线架构，各组件职责清晰，可独立使用也可组合成完整管道：

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户文档集合                              │
│         (PDF, DOCX, XLSX, PPTX, HTML, EPUB, 图片, ...)          │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    文本提取 (ingest/)                            │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐   │
│  │ Kreuzberg   │─▶│ 可选 OCR    │─▶│ 分块 (chunking 10k字符)  │   │
│  │ (75+格式)   │  │ Tesseract   │  │                         │   │
│  └─────────────┘  └─────────────┘  └─────────────────────────┘   │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    模式发现 (schema-free 模式)                    │
│              LLM 从文档样本自动设计实体/关系类型                  │
│              结果保存为 discovered_domain.yaml                    │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    实体关系抽取 (extract/)                        │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │  异步并发处理多个块，每个块一次LLM调用抽取实体+关系          │  │
│  │  支持任意LiteLLM提供商                                      │  │
│  └──────────────────────────────────────────────────────────┘  │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                     知识图谱构建 (graph/)                         │
│  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐   │
│  │  预去重         │─▶│  添加节点和边   │─▶│ 社区发现(Louvain)│   │
│  │ SemHash模糊匹配 │  │ NetworkX存储    │  │                 │   │
│  └─────────────────┘  └─────────────────┘  └─────────────────┘   │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    实体解析 (resolve/)                            │
│  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐   │
│  │ LLM提出合并建议 │─▶│ 人工交互式审核  │─▶│  图手术应用合并  │   │
│  │ 可选embedding聚类│  │ YAML编辑可选   │  │  重连所有边      │   │
│  └─────────────────┘  └─────────────────┘  └─────────────────┘   │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    叙事生成 (narrate/)                            │
│         LLM 生成社区标签、实体描述、 prose 报告                    │
└────────────┬──────────────────────────────────────────────────────┘
             │
             ▼
┌─────────────────────────────────────────────────────────────────┐
│               可视化 / 导出                                        │
│  ┌─────────────────┐  ┌─────────────────────────────────────┐    │
│  │ 浏览器交互式查看 │  │ GraphML/GEXF/SQLite/CSV/JSON导出     │    │
│  │  (pyvis)        │  │                                     │    │
│  └─────────────────┘  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────────┘
```

**项目目录结构**:

```
sift-kg/
├── src/sift_kg/              # 主源代码
│   ├── __init__.py           # 包导出和版本信息
│   ├── __main__.py           # 模块执行入口
│   ├── cli.py                # CLI 接口 (Typer)
│   ├── config.py             # 配置管理 (pydantic-settings)
│   ├── pipeline.py           # 库API函数
│   ├── extract/              # 实体关系抽取
│   │   ├── __init__.py
│   │   ├── models.py         # 抽取结果数据模型
│   │   ├── extractor.py      # 抽取编排（异步并发）
│   │   ├── llm_client.py     # LLM客户端封装 (LiteLLM)
│   │   └── prompts.py        # 提示词构造
│   ├── domains/              # 域配置系统
│   │   ├── __init__.py
│   │   ├── models.py         # 域配置模型
│   │   ├── loader.py         # 域加载器
│   │   ├── discovery.py     # schema-free 模式发现
│   │   └── bundled/          # 预打包域
│   │       ├── general/      # 通用域
│   │       ├── academic/     # 学术文献域
│   │       ├── osint/        # 开源调查域
│   │       └── schema-free/  # 自动发现域模板
│   ├── graph/                # 知识图谱核心
│   │   ├── __init__.py
│   │   ├── knowledge_graph.py # NetworkX封装
│   │   ├── builder.py        # 图谱构建
│   │   ├── prededup.py       # 预去重
│   │   ├── postprocessor.py  # 后处理
│   │   └── communities.py    # 社区检测
│   ├── resolve/              # 实体解析与去重
│   │   ├── __init__.py
│   │   ├── models.py         # 合并建议模型
│   │   ├── resolver.py       # LLM合并建议生成
│   │   ├── clustering.py     # embedding聚类（可选）
│   │   ├── engine.py         # 图手术引擎
│   │   ├── reviewer.py       # 交互式审核界面
│   │   └── io.py             # I/O操作
│   ├── ingest/               # 文档摄取
│   │   ├── __init__.py
│   │   ├── base.py           # 提取后端协议
│   │   ├── reader.py         # 文档读取编排
│   │   ├── chunker.py        # 文本分块
│   │   ├── ocr.py            # OCR抽象
│   │   ├── kreuzberg_extractor.py # Kreuzberg后端
│   │   └── pdfplumber_extractor.py # PDFPlumber后端（遗留）
│   ├── narrate/              # 叙事生成
│   │   ├── __init__.py
│   │   ├── generator.py      # LLM生成
│   │   └── prompts.py        # 提示词模板
│   ├── visualize.py          # 交互式可视化
│   └── export.py             # 多格式导出
├── tests/                    # 测试套件
├── examples/                 # 示例项目 (FTX, Epstein, Transformers)
├── .agents/skills/sift-kg/   # AI agent 集成技能
└── pyproject.toml            # 项目配置
```
